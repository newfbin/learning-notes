### 查看/修改量测数据

> -D：日期
>
> -HM：时分
>
> -C：量测code
> ![image-20260325092614228](./assets/业务工具&命令/image-20260325092614228.png)
>
> -V：要设置的值

**查看量测值**：

```sh
x7fun showhis -D10 -HM0831 -C114560489249351504
```

**修改量测值**：

```sh
x7fun sethis -D10 -HM0831 -C114560489249351504 -V1
```
