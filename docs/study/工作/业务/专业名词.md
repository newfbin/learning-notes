## **开关**（断路器 Circuit Breaker）

1. 主要用于**正常负荷电流的接通与分断**，以及在发生**短路或过载等故障时自动切断电路**，具有保护功能。
2. 能够**带负荷操作**（即在有电流流过的情况下进行分合闸操作）。
3. 具备灭弧装置，能安全地切断大电流。

## **刀闸**（隔离开关 / 闸刀   Disconnector / Isolator）

1. 主要用于**隔离电源**，形成明显的断开点，确保检修人员的安全。
2. **不能带负荷操作**（严禁在有负载电流时拉合），只能在无电流或极小电流（如电容电流）下操作。
3. **没有灭弧能力**，不具备保护功能。

## 容器

在电力系统数字化模型（如 CIM/IEC 61970）中，**“容器”是指可以包含其他设备或子容器的逻辑/物理区域**。它的作用是：

> **组织设备、表达归属关系、支持分层管理与可视化。**

常见的容器类型包括（按层级从大到小）：

| 容器类型                          | 英文                | 典型包含内容                         | 是否可嵌套      |
| --------------------------------- | ------------------- | ------------------------------------ | --------------- |
| **电网（Power System Resource）** | PowerSystemResource | 多个变电站                           | ✅               |
| **变电站（Substation）**          | Substation          | 电压等级、间隔、开关站               | ✅               |
| **开关站（Switching Station）**   | SwitchingStation    | 母线、开关、刀闸、线路终端           | ❌ 或有限嵌套    |
| **间隔（Bay）**                   | Bay                 | 一组功能相关的设备（如一条出线间隔） | ❌               |
| **电压等级（VoltageLevel）**      | VoltageLevel        | 同一电压下的母线和设备               | ✅（属于变电站） |

## 开关站

**变电站**和**开关站**的区别

| 特性             | **变电站（Substation）**               | **开关站（Switching Station）**                          |
| ---------------- | -------------------------------------- | -------------------------------------------------------- |
| **是否含变压器** | ✅ 通常包含主变（电压变换）             | ❌ **不含变压器**，只进行开关操作                         |
| **主要功能**     | 电压变换、电能分配、保护控制           | 线路分段、联络、隔离、转供                               |
| **电压等级**     | 至少两个电压等级（如220/110/10kV）     | **单一电压等级**（如全是110kV）                          |
| **典型设备**     | 主变、各侧开关、母线、无功补偿、站用电 | 母线、断路器、刀闸、线路终端、PT/CT                      |
| **是否独立建站** | 是，有完整站控系统                     | 可能是户外构架，无主控楼                                 |
| **别名**         | —                                      | **开闭所（Distribution Switching Station）**、**开关所** |

## 间隔（Bay）

- 功能单元，如“#1 主变间隔”、“10kV 出线3间隔”；
- 包含：断路器 + 两侧刀闸 + 接地刀 + CT/PT + 电缆终端；
- 用于保护配置、检修隔离、图形绘制。

## 电压等级（Voltage Level）

- 属于变电站的子容器；
- 所有同电压设备归入同一电压等级；
- 例如：变电站下的 “110kV 电压等级” 容器。
