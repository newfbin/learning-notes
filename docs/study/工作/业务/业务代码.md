## SQL语句

### 变压器各侧开关刀闸记录

```sql
SELECT * 
FROM TX7_DEV_ZWBYQKG T 
WHERE T.OFBYQ ='117093592241017162'; -- 变压器设备id
```

### 变压器绕组数量

> 和上面的“获取变压器各侧开关刀闸记录”的SQL语句基本一样，说明可以通过变压器各侧有几个刀闸/开关，来判断变压器有几个绕组

```sql
SELECT COUNT(1) 
FROM TX7_DEV_ZWBYQKG T 
WHERE T.OFBYQ ='117093592241017162'; -- 变压器设备id
```

### 判断变电站是不是用户站

> 存在于 `TOPO_ERROR_USER_STATION` 表中的变电站就是用户站

```sql
SELECT * 
FROM TOPO_ERROR_USER_STATION 
WHERE SS_ID = '113997367497196834' -- 变电站id
```

### 配网变压器（配变）所属的馈线

```sql
select T.URI,T.DESCRIPTION ,A.FDDESC,A.SSDESC,A.KGDESC 
from CIM_POWERTF T,TX7_PWPJ_FEEDER A
where T.OFTOPEQ =A.FDCODE
order by A.SSDESC,A.KGDDNO
```
